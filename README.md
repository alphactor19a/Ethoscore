# Ethoscore Landing Page

A modern, responsive landing page for Ethoscore - discover the hidden bias patterns and rhetorical framing in your news.

## Features

- Clean, minimal design matching Figma specifications
- Responsive layout (1304 x 6926 px design)
- Custom illustrations and graphics
- Multiple sections:
  - Hero section with call-to-action
  - Top protocols showcase
  - Customer testimonials
  - Battle-tested governor section
  - Scalability features
  - Capabilities grid
  - Open source section
  - Product tiers
  - Footer with links

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

To create a production build:

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Design

The design is based on Figma specifications with:
- Custom color palette using CSS variables
- SF Pro Text and Inter fonts
- Precise spacing and typography
- Custom illustrations (windmill, trees, governance charts, garden)

## Technologies

- React 18
- CSS-in-JS with inline styles
- Custom CSS variables for theming
